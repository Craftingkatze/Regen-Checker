# Regen-Checker

Dieses Projekt zeigt eine **Live-Regen-Vorhersage für Deutschland** an. Nutzer:innen können ihre Stadt oder einen Stadtteil auswählen und erhalten aktuelle Wetterinformationen sowie eine **7-Tage-Stundenübersicht** mit Farbstatus.

---

## Statusfarben

- **Rot** – Es regnet jetzt oder es wird innerhalb der nächsten 4 Stunden Regen geben  
- **Gelb** – Es hat kürzlich geregnet oder die Luft ist sehr feucht (>85% Luftfeuchtigkeit)  
- **Grün** – Trocken, kein Regen in den nächsten 4 Stunden erwartet  

---

## Features

- Autovervollständigung für **Städte und Stadtteile Deutschlands**  
- Anzeige von **aktueller Temperatur, Luftfeuchtigkeit und Regenstatus**  
- **7-Tage-Timeline** mit stündlicher Vorhersage  
- Farbvisualisierung für einfache Erkennung von Regen, Feuchtigkeit und Trockenheit  
- Mobile-friendly Design, optimiert für verschiedene Bildschirmgrößen
